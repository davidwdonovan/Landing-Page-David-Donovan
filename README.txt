# Personal Landing Page (v4 — AI + Net Zero)

## Files
- `index.html` — landing page
- `styles.css` — styling
- `script.js` — light JS (year + reveal + reduced-motion handling)
- `assets/headshot.jpg` — your photo (included)
- `assets/circuit.jpg` — background (included)
- `CCS_Chain_Optimizer_White_Paper.docx` — CCS Chain Optimizer white paper (included)
- `resume.pdf` — add your CV in the same folder and name it `resume.pdf`

## Preview
Unzip → double-click `index.html`

## Publish (recommended: Netlify)
1. Create a free Netlify account.
2. Drag-and-drop the folder contents (or the zip) into Netlify “Deploy”.
3. Netlify provides a live URL immediately.
4. Optional: connect a custom domain.

## What to edit
- Replace the LinkedIn URL in `index.html` (search for “Add your LinkedIn URL here”).
- Drop your CV into the folder and name it `resume.pdf`.
